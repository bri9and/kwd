# Conley Resort & Golf Club — Images Needed

All images should be placed in `/public/sites/conley-resort/`.

## Required Images

| Filename | Description | Recommended Size |
|----------|-------------|-----------------|
| `logo.png` | Conley Resort & Golf Club logo (transparent PNG) | 300x180px |
| `hero.jpg` | Aerial or wide shot showing the full resort — golf course, hotel, waterpark visible | 1920x1080px |
| `course1.jpg` | Fairway view — scenic shot of the golf course | 1200x675px |
| `course2.jpg` | Pond/water feature hole — showcasing water hazards | 1200x675px |
| `course3.jpg` | Green with bunkers — approach shot perspective | 1200x675px |
| `course4.jpg` | Tree-lined fairway — showing mature hardwoods | 1200x675px |
| `resort.jpg` | Resort exterior or banquet facility — used as events section background | 1920x1080px |
| `restaurant.jpg` | Restaurant/dining interior or bar area | 1920x1080px |
| `waterpark.jpg` | Waterpark slides or pool area (not currently used in page but reserved) | 1200x675px |

## Notes

- All JPGs should be optimized for web (80% quality, progressive)
- Hero and background images (hero.jpg, resort.jpg, restaurant.jpg) need dark overlay compatibility — avoid very bright or washed-out images
- Logo should have transparent background for use on both dark and light sections
- Course photos are displayed in a 2x2 grid at roughly 600x288px per cell but should be provided at 2x for retina
