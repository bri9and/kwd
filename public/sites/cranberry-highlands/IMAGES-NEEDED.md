# Cranberry Highlands Golf Course — Images Needed

All images go in `/public/sites/cranberry-highlands/`.

## Required Files

| File | Usage | Recommended Size | Notes |
|------|-------|-----------------|-------|
| `logo.png` | Top bar, hero, footer | 300x180px (transparent PNG) | Course logo or wordmark |
| `hero.jpg` | Hero section background + events section | 1920x1080px | Aerial or wide-angle of a signature bunker-laden fairway |
| `course1.jpg` | Photo gallery (top-left) | 1200x675px | Fairway and bunker view — show the sand |
| `course2.jpg` | Photo gallery (top-right) | 1200x675px | Green complex with surrounding sand traps |
| `course3.jpg` | Photo gallery (bottom-left) | 1200x675px | Tree-lined fairway, course beauty shot |
| `course4.jpg` | Photo gallery (bottom-right) | 1200x675px | Signature bunker hole — dramatic sand |
| `clubhouse.jpg` | Clubhouse section background | 1920x1080px | Exterior or interior of the clubhouse/banquet space |

## Image Guidelines

- All JPGs should be optimized for web (80-85% quality)
- Hero and section backgrounds need to work with dark overlays — avoid overly bright/washed-out images
- Course photos should showcase the 90+ bunker identity — sand should be visible and prominent
- Logo should have a transparent background for use on dark headers
- Recommended: include at least one photo showing the scale of bunker complexes
